Sistema de Control de Notas - MVC + PHP + Bootstrap

1. Copia esta carpeta a tu servidor local (htdocs o www).
2. Crea una base de datos llamada HBC en phpMyAdmin.
3. Importa el archivo bsgrupo4.sql desde la carpeta /database.
4. Configura config/db.php si usás otra contraseña o usuario.
5. Accede vía: http://localhost/SistemaNotas_HBC/public
